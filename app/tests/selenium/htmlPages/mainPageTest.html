Vision

The mission of the project is to develop a testing framework that automatically identifies changes in objects, providing reports on the type and amount of changes and factorizations.

The projects vision is to reduce the time engineers must dedicate to the stage of maintenance and refactoring, specifically on automatic and manual tests by using a Self-healing system.

  • Objective 1: Develop an algorithm of change detection on objects in the test cases.
  • Objective 2: Develop a system of reports to show the amount of elements modified.
  • Objective 3: Implement an automatic learning mechanism for the self-healing properties of the system.
  • Objective 4: Develop an user interface so that the engineers can review and validate the changes detected.
